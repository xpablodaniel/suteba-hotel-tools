Generador de Fichas (fichaPax)

Automatiza la creación de fichas de check‑in desde CSV

Qué hace

  • Lee y agrupa el CSV por número de voucher.
  • Selecciona al titular (mayor edad) y agrega hasta 3 acompañantes.
  • Rellena la plantilla PDF y exporta una ficha por voucher en fichas/.

Requisitos

Python 3 + dependencias: reportlab y pypdf.

Descargar plantilla PDF CSV ejemplo (MAP) CSV ejemplo (PC)

Instalación y uso (local)

pip install reportlab pypdf
python3 python/fichaPax/llenar_fichas.py

Previsualizar (opcional)

python3 python/fichaPax/previsualizar_fichas.py

Notas

  • Coloca el CSV diario en python/fichaPax/ o pásalo como argumento al script si así está implementado.
  • Las fichas generadas se guardan en python/fichaPax/fichas/.
  • Los scripts deben ejecutarse localmente; no hay ejecución de Python desde el navegador.
